复习c＋＋指针常量与常量指针
const int p ;//指针的指向可以修改，指针指向的值不可以修改
int const a ;//指针的指向不可以修改，指针指向的值可以修改
const int * const s;
两个字符加减是ascii相加减并且返回一个整型
函数值传递不会修改实参，修改实参必须要取内容值
在函数中遍历一个字符串最好通过另外一个字符串间接访问，这样可以不修改原字符串。
